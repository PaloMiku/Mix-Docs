## 快速上手

欢迎使用 Mix Space，以下是基础功能的使用说明。

### 创建第一篇文章

1. 登录管理后台
2. 进入「文稿」页面
3. 点击「新建文稿」
4. 使用富文本编辑器或 Markdown 编写内容
5. 点击发布

### 内容类型

Mix Space 支持三种内容类型：

- **文稿（Posts）**：正式的博客文章，支持分类和标签
- **手记（Notes）**：简短的日常记录，类似微博
- **思考（Thoughts）**：碎片化的想法和灵感

### 管理友链

在后台「数据管理」中可以管理友链，支持自助申请功能。
